Preguntas

  1. ¿Qué es el DOM?
    El DOM (Document Object Model) es una representación estructurada de un documento HTML o XML en forma de árbol, donde cada nodo representa una parte del documento.
    Fuente: MDN Web Docs
  2. ¿Qué es un nodo?
    Un nodo es cualquier unidad dentro del DOM, como un elemento, un atributo o un texto.
    Fuente: MDN Web Docs
  3. ¿Cómo se accede a un elemento del DOM?
    Se puede acceder a un elemento del DOM usando métodos como document.getElementById(), document.querySelector(), entre otros.
    Ejemplo: document.getElementById('miElemento')
  4. ¿Cómo se modifica un elemento del DOM?
    Se puede modificar usando propiedades como innerHTML, textContent o style.
    Ejemplo: document.getElementById('miElemento').textContent = 'Nuevo texto';
  5. ¿Cuál es la diferencia entre getElementBy* y querySelector*?
    getElementBy* selecciona elementos de manera específica por ID o nombre de etiqueta, mientras que querySelector* permite usar selectores de CSS para obtener elementos.
    Ejemplo: document.querySelector('.miClase')
  6. ¿Qué es un evento?
    Un evento es una acción que ocurre en la página, como un clic, una pulsación de tecla o el movimiento del mouse.
  7. ¿Qué es un listener?
    Un listener (o event listener) es una función que "escucha" un evento y ejecuta un código cuando ese evento ocurre.
    Ejemplo: elemento.addEventListener('click', function() { alert('Clic detectado!'); });
  8. ¿Qué es un callback?
    Un callback es una función que se pasa como argumento a otra función para ser ejecutada más tarde.
    Ejemplo: setTimeout(function() { console.log('Ejecutado después de 3 segundos'); }, 3000);
