Estilos CSS

Lineamientos de css para dar estilos a paginas html

Propiedad Descripción
Propiedades básicas de texto
color Cambia el color del texto.
font-size Define el tamaño del texto.
font-weight Establece el grosor de la fuente (por ejemplo: normal, bold).
text-align Alinea el texto (por ejemplo: left, center, right).
text-decoration Añade decoración al texto (none, underline, overline).
Propiedad Descripción
Propiedades básicas de diseño
background-color Define el color de fondo de un elemento.
border Establece el borde de un elemento.
margin Espacio exterior alrededor de un elemento.
padding Espacio interior entre el contenido de un elemento y su borde.
width Ancho de un elemento.
height Altura de un elemento.
Propiedad Descripción
Propiedades de posicionamiento
position Establece el tipo de posicionamiento (static, relative, absolute, fixed).
top Define la posición desde la parte superior.
left Define la posición desde la parte izquierda.
z-index Controla el orden en capas de los elementos (valor más alto al frente).
overflow Gestiona contenido que excede los límites de su contenedor (visible, hidden, scroll).
