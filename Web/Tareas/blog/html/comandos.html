Comandos de navegación
pwd Este comando te muestra en qué parte del directorio estás
cd <RUTA> Este comando nos permite navegar a la ruta que escribas como argumento
cd .. Regresas en un nivel tu posición en el directorio
touch <NOMBRE_ARCHIVO> Este comando nos permite crear un nuevo archivo, con el nombre que pasas como argumento
ls Nos muestra todos los archivos y carpetas de la raíz en donde estamos
ls -a Nos muestra todos los archivos y carpetas incluyendo los archivos ocultos
mkdir "nombre" Crea un nuevo directorio
rm "nombre_archivo.txt" Elimina archivos o carpetas
cat "nombre_raiz.txt" Muestra contenido de un archivo
Comandos de Github
git init Inicia un repositorio local
ssh-keygen -t ed25519 -C "<ssh del repositorio<" Creamos los archivos ssh locales
git clone "<Tú e-mail de github<" Clonamos un repositorio a la raíz donde estamos
git add -A Registra qué archivos nuevos hay en el repositorio para pasarlos a revisión
git commit -m "nombre del cambio" Se guardan los cambios del repositorio
git status Nos muestra el estatus de nuestro repositorio
git push -u origin main Sube los cambios al repositorio en línea
git pull origin main Baja los cambios del repositorio en línea
